## ✏️ 修改联系信息

打开 `index.html`，搜索「←改这里」，替换成你自己的邮箱、电话、站酷链接。

## 🚀 部署上线

### 方式1：GitHub Pages（免费推荐）
1. 在 github.com 创建新仓库（比如 `portfolio`）
2. 把 `portfolio/` 文件夹里的所有内容上传
3. 仓库 Settings → Pages → Source 选 main branch / root
4. 等待2分钟，访问 `https://你的用户名.github.io/portfolio`

### 方式2：直接发送
把整个 `portfolio/` 文件夹压缩，发给HR即可本地打开 `index.html`
